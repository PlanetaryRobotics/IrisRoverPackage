Detailed Description

Class to run MagicDraw plugin from command line

Before running on headless environment, use Xvfb to set up a virtual X11 display server /usr/bin/Xvfb :1 -screen 0 1024x768x24 export DISPLAY=:1

To run the utility, use the script isfpluginexec.sh in the MagicDrawComdPlugin directory
