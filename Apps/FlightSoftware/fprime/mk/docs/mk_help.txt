help:               This text

ac_lvl1 = Build the stage 1 code genereted files (Serializables)

<BUILD>_clean = Clean output files for a particular build
<BUILD>_doc = Build doxygen files for a particular build
<BUILD>_clean_dox = delete generated doxygen files
<BUILD>_sloc = SLOC count for build
